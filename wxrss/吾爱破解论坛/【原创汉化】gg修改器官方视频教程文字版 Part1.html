63.Well, this is my first tutorial and it is about the game "Escape the Ape”

这个游戏使用的是模糊搜索了！！！！！！他想搞的是能量吧。


果汁随着吃的香蕉越多会满上



思路如下：

  1. 游戏开始前做一次模糊搜索

  2. 然后让猴子吃一些香蕉，接着再做一次模糊搜索表示值不相等了。多重复2这个步骤几次。

  3. 如果没吃到香蕉，那么就搜值没改变，和原始值相等。如果游戏重新开始，果汁是空的，那么就选相等。

  4. 重复2和3直到结果少于100个

  5. 如果果汁满了，那么就搜16。这可能是作者研究出来的一个值吧。

  6. 总之就是重复3，5的过程，幸运的话就能搜到。

最终的效果是找到这两个地址后进行锁定操作。果汁就一直是满的。猴子就能一直往上跳。
